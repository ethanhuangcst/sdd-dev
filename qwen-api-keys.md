## 配置到项目

在项目根目录的 `.env` 文件中添加：

```
DASHSCOPE_API_KEY=YOUR API KEY
DASHSCOPE_BASE_URL=https://dashscope.aliyuncs.com/compatible-mode/v1
DASHSCOPE_MODEL=qwen-plus
PORT=3000
```
